# Budget Tracking App 

Prosta aplikacja do śledzenia miesięcznego budżetu.

##  Funkcje

- Wyświetlanie aktualnego budżetu i zakresu
- Możliwość edycji maksymalnego budżetu
- Pasek postępu pokazujący stopień wykorzystania środków

##  Technologie

- HTML5
- CSS3 + LESS
- JavaScript
